# **App Name**: Predictatron

## Core Features:

- Prediction Input: Allow users to submit their predictions for upcoming events or games.
- Automated Scoring: Automatically calculate scores based on the accuracy of user predictions. Automatically score how accurate user predictions were and then the LLM can tool the database so the app shows an updated leaderboard..
- Leaderboard: Display a real-time leaderboard showing the top-performing players.
- User Profiles: Enable users to create profiles and track their prediction history and performance statistics.
- Email Notifications: Send automated weekly progress emails to players.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2) to represent foresight and intelligence.
- Background color: Light gray (#F0F0F0) for a clean and modern look.
- Accent color: Energetic green (#90EE90) for correct prediction feedback.
- Body and headline font: 'Inter', a sans-serif for clarity and modernity.
- Use clear and concise icons to represent different events or games.
- Design a leaderboard layout that is easy to read and understand, highlighting the top players.
- Incorporate subtle animations for updating scores and leaderboard positions.
